# Misheard Helper — как пользоваться

Скрипт помогает превращать строчки песен в «ослышки»: подбирает фонетически
похожие слова с сохранением ритма. Реальные строчки живут только в твоих
файлах на твоей машине.

## Установка (один раз)

```
pip install wordfreq pymorphy3
```

Первый запуск любой команды строит кэш словаря (~30–60 сек), дальше всё мгновенно.

## Три команды

### 1. suggest — кандидаты замен для каждого слова

```
python3 misheard_helper.py suggest "твоя строка"
```

Для каждого слова выводит до 12 кандидатов с тем же числом слогов и похожим
звучанием. Колонки: «звучание» (фонетическая похожесть 0–1) и «гласные»
(совпадение гласных по позициям — это ритм).

### 2. score — проверка готовой пары

```
python3 misheard_helper.py score "оригинал" "твоя ослышка"
```

Выдаёт вердикт: совпадают ли слоги, держится ли ритм гласных, и не слишком ли
похоже/далеко. Целься в фонетическую похожесть **50–80%** — это «сладкая зона»:
правдоподобно, но не угадывается в лоб.

### 3. batch — пакетные заготовки карточек

```
python3 misheard_helper.py batch lines.txt draft.json
```

`lines.txt` — по строке на строчку (строки с `#` игнорируются).
На выходе JSON с полями карточки (real/mis/artist/title) и кандидатами по
каждому слову. Заполняешь `mis`, `artist`, `title` — и файл готов к импорту в игру.

## Как собирать хорошую ослышку (методика)

1. **Слоги 1:1.** Количество слогов ослышки = количеству слогов оригинала.
2. **Гласные держат ритм.** Меняй согласные смело, гласные — осторожно.
   Особенно важна последняя гласная слова (на неё чаще падает ударение в песне).
3. **Смысл — бытовой абсурд.** Лучшие ослышки звучат как обрывок чьей-то
   странной реальной жизни: еда, соседи, транспорт, животные.
4. **Не бери однокоренные** — скрипт их уже отсекает, но и вручную не подставляй:
   угадывается мгновенно.
5. **Тест вслух.** Прочитай ослышку ровным голосом. Если человек рядом
   морщится и говорит «это же почти как...» — идеально.

## Служебное

```
python3 misheard_helper.py rebuild-cache   # пересобрать словарь кандидатов
```

Кэш лежит рядом со скриптом в файле `.misheard_lexicon.pkl` — можно удалять.
